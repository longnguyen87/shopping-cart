This folder contains files that allowed connection to mylab MongoDB.
file Credentials.js contained login information to mylab MongoDB.
file dbConnection.js established connection to mylab MongoDB. 
file cart-model.js defined various mongoose model that will be used for this project.

productSchema: model that defined each product
itemInCart: model that defined each item in cart. If in the shopping cart has 2 different items, there will be 2 separate instance of this model.
userSchema : model that represent user.
